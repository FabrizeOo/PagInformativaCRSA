# Colegio San Alfonso - Prototipo Informativo

Estructura del sitio web informativo para el Colegio San Alfonso, basada en el diseño del prototipo entregado.

## Secciones Principales
- **Inicio**: Hero section y propuesta educativa.
- **Nosotros**: Reseña histórica y pilares.
- **Niveles**: Educación Inicial, Primaria y Secundaria.
- **Admisión**: Información para el proceso 2026.
- **Noticias**: Actualidad institucional y eventos.
- **Galería**: Registros visuales de actividades y campus.
- **Contacto**: Información de contacto y redes sociales.

## Carpeta de Activos
- `assets/images/`: Para logos, banners y fotos.
- `assets/docs/`: Para calendarios, manuales y reglamentos.
- `css/`: Estilos CSS personalizados.
- `js/`: Scripts de interactividad.
- `pages/`: Estructura modular de las páginas secundarias.
